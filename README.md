## üì° The Core Philosophy

This site rejects standard templates in favor of **Structural Brutalism**:
* **Massive Typography:** Content is treated as architecture.
* **The Void:** High-contrast monochrome aesthetics to reduce cognitive load.
* **Motion Physics:** Interactions that feel weighted and physical, not just animated.

## üìÇ Key Architecture

### 1. The "Anti-Card" Project System
Replaces standard grid layouts with massive, cinematic image strips.
* **Behavior:** Monochrome by default, full-color saturation on hover.
* **Tech:** CSS filters + hardware-accelerated transitions.

### 2. "Initiate Protocol" Contact Form
A terminal-inspired communication interface.
* **Design:** Minimalist line inputs with "Focus" states.
* **Logic:** Connects directly to a Supabase Edge Function to route inquiries via Resend.

### 3. Performance Optimization
* Lazy-loading logic for the abstract video backgrounds.
* Font optimization for large-scale headers (Geist / Inter).
